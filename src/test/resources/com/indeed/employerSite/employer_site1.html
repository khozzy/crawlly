Java Developer – Web Services

Salary Neg Contract Type Contract
Location Dublin Updated 13/09/2013 13:29:48

Java Developer

IT Alliance, an IT Services Provider, are looking to hire a Java Developer with 3+ years’ experience for initial contract roles. The successful candidates will be responsible for maintaining and developing Java and J2EE Applications. This is an excellent opportunity to work with the latest Java and Web technologies.

Role:

•Java and J2EE Development for new and existing applications.
•Web services development.
•Coding in accordance to the standards of the applications.
•Ensure that applications are delivered on time.
•Identifying bugs and resolution.
•Maintaining and repairing applications in order to keep them performing according to technical and functional specifications.
•Unit testing of new applications including the production of detailed test plans.

Requirements:

•At least 3 years Java Development experience in a J2EE environment.
•Development and deployment of JSP and EJB 2 and 3.
•Experience with HTML and XML.
•Web services development experience using XSLT, WSDL and XSD.
•Experience with JavaScript and CSS.
•Oracle Application Server using 9i or higher.
•Experience with Oracle database 10g or higher.
•Strong understanding of OOAD methodologies.
•Must be motivated and have excellent communication skills, both verbal and written.
